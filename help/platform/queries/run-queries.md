# Eseguire query

Questo video mostra come eseguire query nell’interfaccia di Adobe Experience Platform e in un client PSQL. Inoltre, è possibile utilizzare singole proprietà in un oggetto XDM, utilizzare funzioni definite da Adobe e utilizzare CREATE TABLE AS SELECT (CTAS). Per la documentazione dettagliata del prodotto, consulta la [Guida dell&#39;interfaccia utente di Query Editor](https://experienceleague.adobe.com/docs/experience-platform/query/ui/user-guide.html?lang=it).
Per ulteriori informazioni, visitare la [documentazione di Query Service](https://experienceleague.adobe.com/docs/experience-platform/query/home.html?lang=it).
